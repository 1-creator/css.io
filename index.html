Introduction

CSS stands for Cascading Style Sheets. It gives style to web pages written in HTML and XML by enabling developers to set aspects such as the size, position, and color of a page's elements. It can also be used to make animations.

What You Should Know

Before you begin learning CSS, you should know some basic HTML. You can learn more HTML along with CSS. This artice assumes the reader knows some HTML.

CSS Syntax
A CSS declaration has a property and a value.

background-color: blue

In the above example, background-color is the property and blue is the value.

A selectordetermines which part of the document the style rules apply to.

p { background-color: blue; }

In this example, p is the selector.This style rule would give all the p elements in the document a blue backgound. It is possible to select elements by id and class and to select more than one element at a time.

You can add as many CSS declarations as you need to a style block enclosed by curly braces.

p { background-color: blue; color: yellow; }

The above code would give all the p elements a blue background and yellow text (the color property sets the text color).

Box Model

A CSS element such as a div, an article, or a section, by default takes the shape of a rectangular box. Each box has these four parts.

  • Content. This is the main part of the element.
  • Padding. This is the black space between the content, such as the text or image, and the border. The padding can be set for each side of the element.
  • Border. The outer edge of the element. This can be set to appear as a visible line. The width, color, and style can be set for each side.
  • Margin. The space between the element and the next element.
CSS Shorthand

Instead of

border-top: 10px; border-right: 14px; border-bottom: 20px; border-left: 7px;

one can write

border: 10px 14px 20px 7px;

This shortand is called clockwise notation because it starts with the value for the top of the element and goes around the element in the direction of a clock

Responsive Design

Responsive web design is the practice of designing web sites to change depending on the size of a user's screen. For example, a responsive site might have a navbar to the side of the main content on a full-screen computer monitor, but show it above the main content on smartphone screen.

Responsiveness is achieved by using media queries. Here is an example of a media query.

@media(max-width: 799px){ }

This media query would set how the web page looks on any browser with a viewport narrower than 800 pixels wide. Style rules are added between the curly braces.

Flexbox

Flexbox is a CSS module that makes it simple to design responsive sites with a one-dimensional layout. Objects can be set to shrink to fit a small viewport and grow to fit a larger one.

Grid

The grid is a css module that is useful for designs with a two-dimensional layout. Objects can be laid out in both rows and columns.

animations

CSS can be used to create animations. Here are some aspects of an animation it can set:

  • Whether the animation progresses at a steady speed, or whether it speeds up or slows down.
  • How many times the animation plays.
  • The direction in which an animation plays
  • Whether the animation is playing or paused
